电商后台管理系统 📊
🌟 项目概述
基于SpringBoot+Vue3的简易现代化电商管理平台，采用前后端分离架构，为管理人员提供全流程的电商业务支持。系统包含完整的RBAC权限控制体系，支持多维度数据分析和可视化报表展示。
